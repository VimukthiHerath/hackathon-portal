# Q9: GCD via Recursion

Read two integers `a` and `b`. Print their greatest common divisor,
computed using the recursive Euclidean algorithm
(`gcd(a, 0) = a`, `gcd(a, b) = gcd(b, a % b)`).

## Input
```
<a> <b>
```

## Output
A single integer: `gcd(a, b)`.

## Example
Input:
```
48 18
```
Output:
```
6
```

## How to submit
1. Write your solution in a file named `solution.java` (or `solution.py`
   / `solution.cpp` / `solution.js` — any language works), using
   recursion to compute the GCD.
2. Compress it into a standard `.zip` file.
3. Open a submission issue on the portal, selecting `Q9: GCD via
   Recursion` from the Question Selection dropdown.
