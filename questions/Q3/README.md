# Q3: Double Trouble

Read a single integer from stdin and print double its value.

## Input
```
<integer>
```

## Output
A single line: the input value multiplied by 2.

## Example
Input:
```
5
```
Output:
```
10
```

## How to submit
1. Write your solution in a file named `solution.py`.
2. Compress it into a standard `.zip` file.
3. Open a submission issue on the portal, selecting `Q3: Double Trouble`
   from the Question Selection dropdown.
