# Q1: Target Sum Matcher

Write a Python program that reads a list of space-separated integers on
line 1, and a target integer on line 2. Print `True` if any two **distinct**
items in the list add up to the target, otherwise print `False`.

## Input
```
<space-separated integers>
<target integer>
```

## Output
A single line: `True` or `False`.

## Example
Input:
```
4 2 1 7
9
```
Output:
```
True
```
(`2 + 7 == 9`)
