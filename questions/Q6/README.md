# Q6: Valid Parentheses

Read a single line containing only the characters `(`, `)`, `[`, `]`,
`{`, `}`. Print `true` if every bracket is properly opened and closed in
the right order, `false` otherwise.

## Input
```
<a string made only of ()[]{} characters>
```

## Output
A single line: `true` or `false`.

## Example
Input:
```
{[]}
```
Output:
```
true
```

## How to submit
1. Write your solution in a file named `solution.cpp` (or `solution.py` /
   `solution.java` / `solution.js` — any language works).
2. Compress it into a standard `.zip` file.
3. Open a submission issue on the portal, selecting `Q6: Valid
   Parentheses` from the Question Selection dropdown.
