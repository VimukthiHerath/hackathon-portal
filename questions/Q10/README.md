# Q10: First Non-Repeating Character

Read a single line string. Print the first character that appears
exactly once in the string (scanning left to right). If every character
repeats, print `-1`.

## Input
```
<a string with no spaces>
```

## Output
A single character (the first non-repeating one), or `-1` if none exists.

## Example
Input:
```
swiss
```
Output:
```
w
```
(`s` appears 3 times, `w` is the first character that only appears once)
