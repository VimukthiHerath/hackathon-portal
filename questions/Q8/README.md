# Q8: Reverse a Linked List

Read an integer `n`, then a line of `n` space-separated integers. Build
a singly linked list from those values (in the given order), reverse
it, and print the values in their new (reversed) order, space-separated.

## Input
```
<n>
<n space-separated integers>
```

## Output
A single line: the `n` values in reversed order, space-separated.

## Example
Input:
```
5
1 2 3 4 5
```
Output:
```
5 4 3 2 1
```

## How to submit
1. Write your solution in a file named `solution.java` (or `solution.py`
   / `solution.cpp` / `solution.js` — any language works).
2. Compress it into a standard `.zip` file.
3. Open a submission issue on the portal, selecting `Q8: Reverse a Linked
   List` from the Question Selection dropdown.
