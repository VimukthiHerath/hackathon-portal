# Q2: Bubble Sort

Read a single line of space-separated integers from stdin. Print them
sorted in ascending order, space-separated, on one line.

## Input
```
<space-separated integers>
```

## Output
A single line: the same integers, sorted ascending, space-separated.

## Example
Input:
```
5 3 8 1 9 2
```
Output:
```
1 2 3 5 8 9
```

## How to submit
1. Write your solution in a file named `solution.py`.
2. Compress it into a standard `.zip` file.
3. Open a submission issue on the portal, selecting `Q2: Bubble Sort`
   from the Question Selection dropdown.
