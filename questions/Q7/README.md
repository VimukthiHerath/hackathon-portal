# Q7: Binary Search

Read an integer `n`, then a line of `n` space-separated integers already
sorted in ascending order, then a target integer. Print the 0-based
index of the target in the array, or `-1` if it isn't present.

## Input
```
<n>
<n space-separated sorted integers>
<target>
```

## Output
A single integer: the index of `target`, or `-1` if not found.

## Example
Input:
```
6
1 3 5 7 9 11
7
```
Output:
```
3
```

## How to submit
1. Write your solution in a file named `solution.cpp` (or `solution.py` /
   `solution.java` / `solution.js` — any language works).
2. Compress it into a standard `.zip` file.
3. Open a submission issue on the portal, selecting `Q7: Binary Search`
   from the Question Selection dropdown.
