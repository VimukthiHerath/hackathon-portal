# Open Hack

Welcome to **Open Hack** — a live algorithmic contest. Solve the unlocked
problems, submit your solution through a GitHub Issue, and watch the
leaderboard update itself. No manual grading, no separate accounts —
just a GitHub account and this repo.

This repository *is* the contest: the live site, the problem statements,
and the starter packs all live here.

## Get started

- **[Open the portal →](https://vimukthiherath.github.io/hackathon-portal/)**
- [Browse the questions](https://vimukthiherath.github.io/hackathon-portal/questions.html) — locked problems stay listed so you know what's coming.
- [Check the leaderboard](https://vimukthiherath.github.io/hackathon-portal/leaderboard.html) — updates automatically, no refresh needed.
- [Read the full submission guide](https://vimukthiherath.github.io/hackathon-portal/submit.html)

## How it works

1. **Download** the starter pack for the question you're solving, from the Questions page.
2. **Solve it locally**, using the sample input/output in the problem statement.
3. **Zip** your solution file.
4. **Open a submission issue** on this repo, select your question, and attach the zip.
5. **Track your result** — a comment lands on your issue confirming pass/fail and your score, usually within a couple of minutes. Your score appears on the leaderboard right after.
