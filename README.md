# 🏆 SLIIT Hackathon Workspace

Welcome to the challenge! Follow the instructions below to submit your code.

## 📝 Challenge: Target Sum Matcher (ID: Q1)
Write a Python program that reads a list of space-separated integers on Line 1, and a target integer on Line 2. Print `True` if any two distinct numbers add up to the target, otherwise print `False`.

### How to Submit:
1. Write your solution in a file named `solution.py`.
2. Compress `solution.py` into a standard `.zip` folder.
3. Go to the **Issues** tab of this repository.
4. Click **New Issue**, name it `Q1 Submission`, drag and drop your `.zip` file into the description box, and click **Submit**.
